----문제

<pass>부분에 배열 내장함수를 이용하여 코드를 입력하고 다음과 같이 출력되게 하세요.

**데이터**
var arr = [200, 100, 300];
//pass
console.log(arr);

**출력**
[200, 100, 10000, 300]
```




-- 풀이


var arr = [200, 100, 300];
//pass
console.log(arr);
arr.splice(2,0,10000);
console.log(arr);





// 기존 arr 내용 출력
 [200, 100, 300]

// splice() 로  내용 추가해서  푼것.
[200, 100, 10000, 300]
